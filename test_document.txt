GOLFVX 골프카트 관리 시스템 v2.0

골프장 전용 카트 관리 기능:

1. 골프카트 등록 시스템
- 카트 번호: GC001부터 GC100까지 (총 100대)
- 카트 상태: 사용중/대기중/정비중/충전중
- 배터리 타입: 리튬이온 배터리
- 최대 속도: 25km/h

2. 배터리 모니터링 시스템  
- 실시간 배터리 잔량 표시 (0~100%)
- 배터리 온도 모니터링 (정상: 15~35도)
- 충전 시간 예측 기능
- 배터리 교체 알림 (사용량 80% 이상 시)

3. 골프코스 GPS 추적
- 18홀 코스 실시간 위치 추적
- 각 홀별 카트 위치 표시 (1번홀~18번홀)
- 코스 내 위험구역 알림
- 카트 속도 제한 구역 설정

4. 예약 연동 기능
- 골프 예약 시스템과 API 연동
- 예약 30분 전 자동 카트 배정
- VIP 고객 전용 카트 우선 배정
- 라운딩 종료 후 자동 회수

5. 캐디 전용 모바일 앱
- 담당 카트 배터리 상태 확인
- 고객 정보 및 예약 내역 조회
- 긴급상황 신고 버튼
- 카트 위치 실시간 안내

6. 관리자 대시보드
- 전체 100대 카트 현황 모니터링
- 18홀 코스별 카트 분포 현황
- 배터리 충전 스케줄 관리
- 일일 카트 이용 통계
