사용자 관리 시스템 기획서

1. 로그인 기능
- 사용자는 이메일과 비밀번호로 로그인할 수 있다
- 로그인 성공 시 메인 페이지로 이동한다
- 로그인 실패 시 오류 메시지를 표시한다

2. 회원가입 기능
- 사용자는 이메일, 비밀번호, 이름으로 회원가입할 수 있다
- 이메일 중복 확인이 필요하다
- 비밀번호는 8자 이상이어야 한다

3. 프로필 관리
- 사용자는 프로필 정보를 수정할 수 있다
- 프로필 사진을 업로드할 수 있다
- 비밀번호를 변경할 수 있다

4. 게시판 기능
- 사용자는 게시글을 작성할 수 있다
- 게시글 목록을 볼 수 있다
- 게시글을 수정하고 삭제할 수 있다
