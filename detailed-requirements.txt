GOLFVX Fleet Management System v2.0 상세 요구사항서

1. 시스템 개요
- 골프장 카트 및 차량 관리를 위한 웹 기반 시스템
- 실시간 위치 추적 및 배터리 모니터링
- 예약 시스템과 연동된 자동 배차

2. 주요 기능 상세

2.1 골프카트 관리
- 카트 번호: GC001~GC100 (100대)
- 배터리 상태: 충전량, 예상 사용시간, 충전 필요 알림
- 카트 상태: 사용중, 대기중, 정비중, 충전중
- GPS 모듈을 통한 실시간 위치 추적
- 코스별 위치 매핑 (1번홀~18번홀)

2.2 예약 연동 시스템
- 골프 예약 시스템(PMS)과 API 연동
- 예약 시간 30분 전 자동 카트 배정
- 라운딩 종료 후 자동 회수 알림
- VIP 고객 전용 카트 우선 배정

2.3 배터리 모니터링
- 실시간 배터리 잔량 표시 (%)
- 배터리 온도 모니터링
- 충전 이력 관리
- 배터리 교체 주기 예측

2.4 코스 관리
- 18홀 코스 맵 연동
- 각 홀별 카트 위치 실시간 표시
- 코스 내 위험구역 설정 및 알림
- 카트 속도 제한 구역 설정

2.5 관리자 대시보드
- 전체 카트 현황 한눈에 보기
- 배터리 상태 모니터링
- 예약 현황 및 배차 계획
- 일일/주간/월간 운영 보고서

2.6 모바일 앱 (캐디용)
- 담당 카트 상태 확인
- 고객 정보 및 예약 내역
- 긴급상황 신고 기능
- 카트 위치 안내

3. 기술 요구사항
- React.js 프론트엔드
- Node.js/Express 백엔드
- PostgreSQL 데이터베이스
- Redis 캐시 서버
- Socket.io 실시간 통신
- GPS 모듈 API 연동

4. 보안 요구사항
- JWT 토큰 기반 인증
- HTTPS 통신 필수
- 관리자/캐디/고객 권한 분리
- API 요청 제한 (Rate Limiting)

5. 성능 요구사항
- 카트 위치 업데이트: 10초 간격
- 대시보드 로딩 시간: 3초 이내
- 동시 접속자: 최대 50명
- 99.9% 가동률 목표

6. 특수 기능
- 날씨 API 연동으로 우천 시 카트 자동 회수
- AI 기반 카트 배치 최적화
- 카트 충돌 방지 알고리즘
- 분실 카트 자동 탐지 시스템
